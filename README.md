<h1>Did I Watch It</h1>

Did I Watch It is a TV Show tracker app.  

<ul>
<li type="cirle">When the app first starts it will show a list of shows that the user is watching.</li>
<li type="cirle">Pressing the add button the user is able to search and add a show to the watch list.</li>
<li type="cirle">The user can select a show and see all the episodes for the show.</li>
<li type="cirle">The user can select the episode and see the description of the episode and will have the ability to select if the episodes was watched.
<li type="cirle">To change the status of the episode to watched or not watched the user can change the switch state and will receive a overlay with confirmation.
</li>
</ul>
